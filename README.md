# Steam Account Checker

Check for Steam accounts that haven't been set up, level 0 accounts or accounts that have old Valve games but are not active.

## Usage

### Note: You _must_ have Python3 and the `requests` package installed in order to use this script.

- Edit the `Settings.json` file, changing the `start_id` and `end_id` to whatever suits your need and choosing which accounts you want to search by editing the `unverified_accounts`, `lvl0_accounts` and `css_accounts` variables. You also need to set `api_key` to a valid Steam API Key 
- Run the `checker.py` script and wait.
